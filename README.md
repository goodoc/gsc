# Goodoc Sever template cli

## Install
```
npm install -g goodoc-server-cli 
```

## How to use
### Create Project
* Create dir project for new project
```
gsc create [ProjectName]
```

### Add Model
* Append prisma model & add schema
```
gsc model [NewModelName]
```

### Add Resolver
* Add custom resolver
```
gsc resolver [NewResolverName]
```
